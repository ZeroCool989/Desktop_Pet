# Assets

Drop your Meshy AI exports here — **every `.glb` in this folder is loaded**.
They must share the same rig: the first file (alphabetically) is the display
model, and animation clips are merged from all of them. That's how the
Walking + Running Meshy exports combine into one animated panther.

- Clips are auto-mapped by name: idle/breath → idle, talk/roar/growl → talk,
  walk → prowling, run → agitated jog while nagging.
- Anything the clips don't cover falls back to procedural life: breathing,
  head-tracking your cursor, blinking via morph targets, head bob while
  talking. With no .glb at all, a low-poly placeholder renders.

`trayTemplate.png`, `trayTemplate@2x.png` and `tray.png` are generated by
`npm run gen:icons` (also runs on postinstall).
